toString(value) → {string}

Source:
  • toString.js, line 22
Since:

Converts value to a string. An empty string is returned for null and undefined values. The sign of -0 is preserved.

Example
toString(null)
// => ''

toString(-0)
// => '-0'

toString([1, 2, 3])
// => '1,2,3'
